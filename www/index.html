RpepXML

RpepXML is an R package that allows to import and manipulate pepXML files. Latest source code is available on github and the project's summary page.

Getting started

The package can be installed using

install.packages("RpepXML",repos="http://R-Forge.R-project.org")

or for by downloading the package from github. Note that it depends on Biobase (a Bioconductor package), XML and plys.

It can then be loaded with

library(RpepXML)

Documentation

Basic information about the package can be accessed within R with

?RpepXML and help(package="RpepXML")

and all classes, methods and functions are documented. e.g.

?parseMSMSpepXML or ?"MSMSpepXML-class"

More information about the package and its usage can be found in the vignette

vignette(package="RpepXML","RpepXML")

Lates changes can be found with

news(package="RpepXML")
